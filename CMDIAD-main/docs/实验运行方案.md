# PIRN-CMPT 全类实验与消融运行方案

本文最终实验收敛为 1 个主实验和 3 组核心消融。实验围绕三个核心问题组织：

1. 少样本：少量正常样本是否能学习稳定正常原型。
2. 跨模态：测试阶段缺失 RGB 或 SN 时，CMPT 是否能补全伪原型/伪特征。
3. 共享原型：shared prototypes 是否能增强 RGB 与 SN 的共同正常结构表达。

## 统一设置

- 数据集：MVTec 3D-AD 十类。
- 训练数据：仅使用每类正常样本。
- 测试数据：完整测试集，包含正常与异常。
- 默认 K-shot：`K=4`。
- 随机种子：`0, 1, 2`。
- 主指标：Image AUROC、Pixel AUROC、AU-PRO、AU-PRO-0.01。
- 双模态完整运行：`--main_modality ""`。
- RGB 可用、SN 缺失：`--main_modality rgb --allow_true_missing_modality`。
- SN 可用、RGB 缺失：`--main_modality sn --allow_true_missing_modality`。

注意：当前代码不使用 `--main_modality both`，双模态完整必须写成空字符串。

## 最终推荐运行脚本

脚本：

```bash
cd /root/autodl-tmp/PIRN-CMPT/CMDIAD-main

bash scripts/run_thesis_final_15configs.sh
```

调试单类：

```bash
CLASSES=rope SEEDS="0" bash scripts/run_thesis_final_15configs.sh
```

默认设置：

- 十类：`bagel,cable_gland,carrot,cookie,dowel,foam,peach,potato,rope,tire`
- Seed：`0,1,2`
- 主实验：K=16，Full Model，双模态完整
- 少样本：K=1,2,4,8,16，Full Model，双模态完整
- 跨模态缺失：K=16，RGB-only / SN-only / RGB+SN，Full 与 w/o CMPT
- 共享原型：K=16，无原型 / 仅 specific / 仅 shared / specific+shared

注意：脚本为了节省运行量会复用结果：

- 主实验 `K=16 Full Model` 复用少样本实验中的 `K=16`。
- 共享原型中的 `Specific + Shared` 复用 `K=16 Full Model`。

因此论文可写 15 个唯一配置，脚本实际新增运行 13 个配置；复用项不重复跑。

## 单项消融和解释关系

### 少样本

比较不同 `K` 的 Full Model：

```text
K = 1, 2, 4, 8, 16, full
```

回答问题：少样本正常图像是否足够建立稳定原型记忆。

### 跨模态

比较：

```text
RGB-only Full vs RGB-only w/o CMPT
SN-only Full vs SN-only w/o CMPT
```

回答问题：缺失模态时，CMPT 是否提供有效伪模态信息。

### 共享原型

比较：

```text
w/o Shared
only Shared
Specific + Shared
Specific + Shared + pseudo
```

回答问题：共享原型是否学习到 RGB 与 SN 的共同正常结构，且是否需要与模态特定原型互补。
